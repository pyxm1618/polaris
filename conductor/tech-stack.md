# Technology Stack

## 1. Frontend Core
*   **Framework**: **Nuxt.js** (Vue 3)
    *   *Why*: Critical for Baidu SEO (SSR support).
*   **Language**: **TypeScript**
    *   *Why*: Type safety for complex scheduling logic.
*   **Styling**: **Vanilla CSS**
    *   *Why*: To achieve the "Ethereal Glass" (梦幻玻璃拟态) aesthetic with maximum control over `backdrop-filter`, gradients, and micro-animations.

## 2. Backend & Services
*   **Deployment**: **Vercel**
    *   *Why*: Zero-config, fast global CDN.
*   **Database**: **Turso** (Edge SQLite)
    *   *Why*: Hyper-fast edge access, effectively unlimited free tier for personal use.
*   **Authentication**: **Clerk**
    *   *Why*: Specialized auth provider with a beautiful UI that blends well with premium designs. Generous free tier.

## 3. Development Tools
*   **Package Manager**: `pnpm` (Recommended for speed)
*   **Version Control**: Git
